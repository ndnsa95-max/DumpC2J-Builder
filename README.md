# DumpC2J-Builder

> Build scripts, addons & CI workflow for **DumpC2J-Kernel** (Poco F7 / Onyx)

---

## About

This repo handles:
- GitHub Actions CI workflow for automated kernel builds
- AnyKernel3 flashable ZIP packaging
- Toolchain setup & switcher scripts
- Kernel config management
